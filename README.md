# Python-Path-Finder
A* Path finder written in python, uses Octile Heuristic to calculate f Values

## Dependencies
`python 3+`
`pip install pygame`

## How to run
Open main.py

## Usage
Place wall: `W`

Remove wall: `R`

Set start positon: `S`

Set end positon: `E`

Start/Cancel simulation: `SPACE`

Clear Walls: `C`

Toggle Diagonals: `D`

Toggle Grid Lines: `G`
