# Python-Path-Finder
A* Path finder written in python

## Dependencies
`python 3+`
`pip install pygame`

## How to run
Open main.py

## Usage
Place wall: `W`

Remove wall: `R`

Set start positon: `S`

Set end positon: `E`

Start simulation: `SPACE`
